## Topics Covered

### Boolean Values
- True and False values
- Comparison operations (==, !=, <, >, <=, >=)
- Logical operators (and, or, not)
- Truthy and falsy values

### If-Else Statements
- Basic if conditions
- if-else blocks
- Multiple conditions with elif
- Nested if statements
- Shorthand if (ternary operator)
- pass statement

### Loops
- while loops
- for loops
- break statement
- continue statement
- range() function
- else clause with loops
- Nested loops
